### 要求

- 实现web的2048游戏，使用JavaScript或TypeScript完成
- 基本规则、计算分数和判断终局准确无误
- 移动数字时有基本的动画效果

### 加分项

- 适配移动端
- 其他额外的效果或可玩性改进
- 代码实现优雅，注释完整
